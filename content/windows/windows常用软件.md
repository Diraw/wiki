# Ditto

[Releases · sabrogden/Ditto](https://github.com/sabrogden/Ditto/releases)

**使用：**
- 双击或拖动或shift enter粘贴
- 按住ctrl多选

**配置：**
- 常规
	- 窗口弹出位置：上次打开位置
	- 粘贴条目的过期日期：3天
- 支持类型
	- 删除 Rich Text Format（不然复制较大excel表格会卡死）
- 键盘快捷键
	- 激活Ditto：Ctrl + Alt + ,
	- 粘贴时将剪贴项移动到列表顶部
	- 使用当前所选组的最后十个复制项目
		- 位置1：Ctrl + Num 1
		- ……
