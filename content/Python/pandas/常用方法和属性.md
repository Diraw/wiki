# 方法

`df.head(n=5)`：显示 `DataFrame` 的前 `n` 行（默认5行）

`df.tail(n=5)`：后 `n` 行（默认5行）

`df.info()`：打印 `DataFrame` 的简洁摘要，包括每列的非空值数量、数据类型和内存使用情况

`df.describe()`：生成描述性统计信息，包括计数、均值、标准差、最小值、25/50/75百分位数和最大值（仅适用于数值列）

`df.loc[]`：基于标签（索引名和列名）进行选择

`df.iloc[]`：基于整数位置进行选择

`df.max()`：提取每列的最大值

`df.max(axis=1)`：提取每行的最大值

`df.max().max()`：全局最大值

`df.apply(func, axis=0, raw=False, result_type=None)`：沿 `DataFrame` 的轴应用函数（对行或列）

`df.map(func)`：将函数应用于 `Series` 中的每个元素
# 属性

`df.shape`：返回一个元组，表示 `DataFrame` 的维度（行数, 列数）

`df.columns`：返回 `DataFrame` 的列标签（列名）

`df.index`：返回 `DataFrame` 的行索引

`df.dtypes`：返回一个 Series，显示每列的数据类型
