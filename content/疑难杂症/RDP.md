# 2

如果远程是用微软邮箱账户登陆的，用户名直接输入 `邮箱` 可能登不上

**需要在前面加上`MicrosoftAccount\` 才行**

即，用户名输入：

```txt
MicrosoftAccount\xxx@outlook.com
```
